# Margonem - panel dodatkow

Zainstaluj osobno:

1. panel-dodatkow.user.js
2. autox.user.js
3. autoblogo.user.js
4. ulepszarka.user.js
5. auto-kalendarz.user.js

Alternatywnie zainstaluj tylko pakiet.user.js, aby Tampermonkey zaladowal wszystkie moduly automatycznie.

Panel automatycznie wykrywa zainstalowane dodatki i uklada ich ikony pionowo.
Kazdy dodatek mozna aktualizowac i usuwac niezaleznie.

## Publikacja na GitHubie

Wgraj wszystkie pliki do glownego katalogu publicznego repozytorium. Uzytkownik instaluje pliki przez adres Raw.

Po utworzeniu repozytorium mozna dopisac do naglowkow kazdego userscriptu:

```js
// @updateURL   https://raw.githubusercontent.com/LOGIN/REPO/main/NAZWA.user.js
// @downloadURL https://raw.githubusercontent.com/LOGIN/REPO/main/NAZWA.user.js
```
